# Personal Website

This repository contains a static academic-style personal website ready to publish with GitHub Pages.

Main files:

- `index.html`: homepage content and section structure
- `styles.css`: academic visual style and responsive layout
- `assets/profile-photo-placeholder.svg`: placeholder image until you add a real portrait
- `about-me.md`: planning notes for the content you want to fill in

How to customize:

1. Replace `Your Name`, affiliation, email, and short biography in `index.html`.
2. Add your real portrait to `assets/` and update the image path in `index.html`.
3. Edit the education, research, publications, and skills sections with your real information.
4. Push the repository to GitHub and enable Pages from GitHub Actions.

Publishing:

- The site is configured to publish from the root of the `main` branch.
- Because the repository name is `Rice-cas.github.io`, the published URL is the root user site.
